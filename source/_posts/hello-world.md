这是用GitHub的站点，百度了一番教程，做出来的静态blog。用hexo驱动，主题是 hexo官网里的archer。在本地编写md文件后，由hexo编译，再上传到GitHub上，完成博文更新。

## 快速开始

### 新建一篇文章

``` bash
$ hexo new "My New Post"
```

### 启动hexo用于调试测试

``` bash
$ hexo server
```

### 生成静态页面

``` bash
$ hexo generate
```

### 部署到服务端

``` bash
$ hexo deploy
```
